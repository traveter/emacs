* 設定ファイル
  init-loader.elを用いて設定ファイルを分割読み込み
  - 00-init.el
    以下のサイトを参照
    http://d.hatena.ne.jp/jimo1001/20090921/1253525484
  - 01-display.el
    emacsのウィンドウ、フォント、色等の設定
  - 02-encode.el
    エンコードの設定(UTF-8)
  - 03-keybinds.el
    キーバインドのカスタマイズ
  - 04-skk.el
    日本語変換にDDSKKを用いる
    プレフィックスキーの設定
  - 05-common.el
    メジャーモード共通の設定
  - 06-loadelisp.el
    外部elispの読み込み
  - 11-navi2ch_riece.el
  - 14-eshell.el
    emacsのshell実装 eshellの設定
  - 15-c-mode.el
  - 16-php-mode.el
  - 17-java-mode.el
  - 18-smarty-template.el
  - 19-mode.el
  - 20-tex.el
  - 21-maxima.el
  - 22-sql.el
